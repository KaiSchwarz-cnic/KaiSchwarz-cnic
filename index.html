CentralNic WHMCS-based Domain Migration Tool

Bulk Transfers are not supported in WHMCS and there's no Migration Tool available on the market. So, Migrations in WHMCS are bound to a lot effort on reseller-side! This Module helps as far as possible and migrates domain at the point of the domain renewal. It got realized in an generic way allowing portfolio migrations from any registrar in the world to any of the CentralNic Brands (HEXONET, RRPproxy, internet.bs, TTP Wholesale).
This tool is encrypted and kept within a private repository.
